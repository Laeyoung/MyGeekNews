# **App Name**: GeekNews Personal Search

## Core Features:

- Data Fetching and Storage: Scrape and store user's upvoted GeekNews articles from https://news.hada.io/upvoted_topics?userid=laeyoung, handling pagination to retrieve all articles. User ID will be configurable. This feature will be triggered manually.
- Search and Filtering: Provide a search interface with filtering options (e.g., last month, last year) to search through the scraped GeekNews articles.
- Results Display: Display search results in a clear, paginated list with links to the original GeekNews articles.

## Style Guidelines:

- Text: #232323 (Gray tone to match GeekNews).
- Background: #f3f3f3 (Light gray to match GeekNews).
- Highlight: #697070 (Gray highlight color to match GeekNews).
- Clean, card-based layout for search results.
- Simple, recognizable icons for filtering and sorting options.
- Subtle transitions for search results and filter changes.
